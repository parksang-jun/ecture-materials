# 강의자료 — 반짝반짝 원자력 세상!

초등학교 3~4학년 대상 원자력 발표 자료입니다.
한국원자력연구원의 「원자력 표준 강의안」을 초등학생 눈높이에 맞게 재구성했습니다.

## 파일 목록

| 파일 | 설명 |
|---|---|
| `원자력 세상_초등 3-4학년용.pptx` | 발표 슬라이드 (27장, 16:9) — 5코스 탐험 구성, 퀴즈 2회, 유튜브 영상 슬라이드 4장 포함 |
| `원자력 세상_발표 스크립트.md` | 슬라이드별 발표 스크립트 + 발표 전 체크리스트 + 영상 링크 모음 (같은 내용이 PPT 발표자 노트에도 수록) |

## 구성 (5코스 탐험)

1. ⚡ **에너지가 뭐야?** — 밥·자동차·스마트폰 비유, 롤러코스터로 에너지 변신
2. 🔎 **쪼개고 쪼개면? 원자!** — 레고 블록 비유, 핵분열은 도미노로
3. 🏭 **전기를 만드는 마법** — 발전소 = 커다란 주전자
4. 🛡️ **방사선과 안전** — 영화 vs 진짜, 다섯 겹 갑옷(5중 방호벽)
5. 🚀 **미래의 원자력** — 인공태양 KSTAR, 우주 배터리, SMR

## 사용 방법

- 발표 시 PowerPoint **발표자 보기**를 켜면 각 슬라이드의 스크립트가 노트로 표시됩니다.
- 영상 슬라이드(12, 16, 25, 26번)의 카드를 클릭하면 유튜브로 이동합니다.
- 영상 검색 카드에는 유튜브의 **"4분 이내 영상만 보기" 필터**가 걸려 있어 짧고 쉬운 영상만 나옵니다.
- 유튜브 영상은 삭제·변경될 수 있으니 **수업 전날 링크를 한 번씩 확인**하고 보여줄 영상을 미리 골라 두세요.
